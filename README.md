# Empathix

Інтелектуальний помічник для аналізу емоційного забарвлення тексту та автоматизації підбору доцільних виразів.

## Функціонал
- Генерація відповідей за допомогою AI з урахуванням обраного тону.
- Використання переліку `Tone` для градації стилів спілкування.
- Легка інтеграція через консольний інтерфейс.

## Технічні характеристики
- **Платформа**: .NET 8.0
- **Мова розробки**: C#
- **Інструменти**: Visual Studio Code 1.110, Git 2.47, GitHub, GitHub Copilot Student

## Інструкція з використання
